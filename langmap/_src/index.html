[WIP] langmap: IPA geographic visualizer in GLSL

Note: This is not guaranteed to be a correct representation of the incorporated datasets yet, I need to double check the data processing. Entirely possible things are flipped or scaled incorrectly.

Click buttons to locate the origin places of languages that use these phonemes.

Map projection: Mercator

Tools used: aalib.js for image->ASCII conversion, xemantic/shader-web-background for GLSL fragment shading

Datasets used: Phoible and Glottolog

Also referenced this helpful guide on CSS mix-blend-mode to use a text layer as a mask for a layer beneath.
