langmap: IPA geographic visualizer in GLSL

Click buttons to locate the origin places of languages that use these phonemes.

Caveat: This is not guaranteed to be a correctly scaled representation of the incorporated datasets yet, I need to double check the data processing and map projection equations.

Map projection: Mercator

Datasets used: Phoible and Glottolog. Note that this visualization does not accurately represent where phonemes are spoken, only where they're logged as originating from within a corpus of languages focused on rare dialects. Since Glottolog is geared toward niche languages, I don't have geolocational data on a region to which a language may have traveled, only a single lat/long coordinate.

The ASCII+shader map display builds off my previous drabble, skymap.
